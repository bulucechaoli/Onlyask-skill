# 使用示例

## 输入示例 1：首次使用

```
用户：我想用苏格拉底教学法学习这本《人工智能：现代方法》的教材。
      教材路径：/docs/aima.pdf
      我的昵称是小明，计算机专业大三学生。
      我想设定三个角色：
      1. 艾达·洛芙莱斯 - 严谨优雅，喜欢从数学原理出发讲解
      2. 图灵 - 幽默风趣，喜欢用类比和思想实验
      3. 香农 - 务实高效，喜欢从工程应用角度切入
```

## 输出示例 1：系统初始化

```
AI：(初始化项目结构)
   创建 socratic-teacher/ 目录及所有模板文件
   分析教材目录结构
   写入角色人设
   生成进度表

AI：系统已初始化完成！

   📚 教材：《人工智能：现代方法》共 27 章
   👤 学习者：小明（计算机专业大三）
   🎭 虚拟老师：
      - 艾达·洛芙莱斯（严谨·数学派）
      - 图灵（幽默·思想实验派）
      - 香农（务实·工程派）

   请告诉我你想从第几章开始学习，以及由哪位老师来授课？
```

## 输入示例 2：日常上课

```
用户：今天从第1章开始，让艾达来教吧
```

## 输出示例 2：苏格拉底式教学

```
艾达·洛芙莱斯：*整理了一下裙摆，坐到你对面，翻开一本笔记*

  很高兴今天能和你一起开始这段旅程。在翻开第一章之前，
  我想先问你一个问题——

  你觉得，什么是"智能"？
  不要查书，也不用给标准答案。告诉我你自己的直觉。
```

## 输入示例 3：下课

```
用户：下课了
```

## 输出示例 3：课后更新

```
AI：(自动更新 progress.md)
   (自动更新 book_revision_notes.md)
   (自动更新 diary.md)
   (自动更新 wechat_unread.md)

艾达·洛芙莱斯：*合上笔记，微微一笑*
  今天学得很认真呢。期待下次见面。

  ===== 课后更新完成 =====
  - 进度已保存至 progress.md
  - 日记已更新至 diary.md
  - 微信群有 3 条新消息，输入"我想看看微信群聊"查看
```
